Post Processing Profiles 3.1


If you use Unity 2017:
Unzip files from PostProcessing Profiles/PP/PostProcessing1.zip to Assets folder for v1 profiles.
Unzip files from PostProcessing Profiles/PP/PostProcessing2.zip to Assets folder for v2 profiles.

If you use Unity 2018 with Package Manager:
Please import Post processing-2 from Unity Package Manager for v2 and v3 profiles., after this - import Post Processing profiles.
(https://docs.unity3d.com/Packages/com.unity.package-manager-ui@1.8/manual/index.html)



Version 3 - How to use:
Move Blood, Camera1, Camera_BW, Ice prefab to your scene.
Or move ALL_OtherProfiles prefab to your scene, after this add profile to Post Process Volume (script) -> Profile.

Version 1 and 2 - How to use:
Move Profile prefab to your scene.
